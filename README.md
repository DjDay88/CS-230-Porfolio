# CS-230 Portfolio

## Summary

The client for this project was The Gaming Room. They wanted to expand their game “Draw It or Lose It” into a web based application that can run across multiple platforms. The system supports multiple players and teams and is designed to be scalable, secure, and run in a distributed environment.

## Reflection

I created a software design document that includes requirements, design constraints, system architecture, a UML diagram, evaluation, and recommendations. This project helped me understand how important planning is before coding and how different platforms affect system design.
